Este projeto tem como objetivo desenvolver um sistema de agendamento de consultas genérico, destinado a diversas áreas de atendimento, visando solucionar os desafios de gerenciamento de horários e organização dos atendimentos. Utilizando técnicas avançadas de programação, buscamos criar um guia claro e acessível que incorpore conceitos complexos de maneira compreensível, facilitando a análise e o desenvolvimento de um sistema de agendamento eficiente e intuitivo, especialmente para clínicas e consultórios.
Para atingir esse objetivo, aplicaremos os princípios fundamentais da engenharia de software. Iniciaremos com uma análise detalhada dos requisitos funcionais e não funcionais necessários ao sistema de agendamento, juntamente com a modelagem de sistemas usando a linguagem UML . Através de diagramas de classes, ofereceremos uma representação visual precisa e abrangente do processo .
Para criar esses diagramas, utilizaremos a ferramenta Draw.io . A implementação do sistema será dividida em duas partes: a primeira parte abrange o Front-End, utilizando HTML, CSS e JavaScript . A segunda parte, referente ao Back-End, empregará SQL e JAVA . Esta divisão visa melhorar a interação entre as partes e assegurar que todos compreendam seu papel no desenvolvimento do sistema.
Escolhemos essas linguagens devido à sua acessibilidade e capacidade de criar interfaces visuais atraentes e páginas web dinâmicas . Utilizando o ambiente de desenvolvimento Visual Studio, garantiremos uma abordagem coesa e bem-sucedida .
Além disso, este projeto explorará os fundamentos da engenharia de software, apresentando definições técnicas relevantes e contextualizando a importância da programação de forma geral . Nosso objetivo final é proporcionar uma compreensão mais profunda do processo de desenvolvimento de um sistema genérico de agendamento de consultas, assim como dos conceitos fundamentais da engenharia de software e programação. Este projeto será essencial para construir uma aplicação web com banco de dados que facilite o gerenciamento de horários e atendimentos, oferecendo uma plataforma prática para usuários organizarem suas consultas e compromissos com mais eficiência .
